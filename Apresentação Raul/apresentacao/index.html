Módulo 2 — Data Science com IA

Web Scraping com IA

Quando os dados nao estao em formato acessível, a IA ajuda a extrair

Processo com IA

  1. Identificar a fonte de dados (site, PDF, API)
  2. Pedir a IA para gerar o código de extração
  3. A IA escreve o scraper em Python
  4. Executa, coleta e organiza os dados
  5. Exporta para CSV/Excel para análise

Exemplo: Extrair precos da ANP

# Codigo gerado pela IA automaticamente
import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "https://www.gov.br/anp/..."
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

# Extrair tabela de precos
tabela = soup.find('table', class_='dados')
df = pd.read_html(str(tabela))[0]

# Salvar em Excel
df.to_excel('precos_anp.xlsx', index=False)
print(f"Extraidos {len(df)} registros")
Curso ABAR 2026 15 / 28
Web scraping e essencial quando os dados nao estao em CSV/API. A IA gera o código completo — o regulador so precisa dizer o que quer. Exemplo real: extrair precos de combustiveis do site da ANP.
Módulo 2 — Data Science com IA

Limpeza de Dados com IA

A etapa mais demorada — a IA reduz de horas para minutos

Problemas comuns

⚠ Valores ausentes (NaN, vazios)

⚠ Formatos inconsistentes (datas, moedas)

⚠ Duplicatas e erros de digitacao

⚠ Outliers e valores impossiveis

⚠ Encoding (acentos, caracteres especiais)

Solucao: prompt para a IA

Prompt: "Limpe este dataset: - Remova duplicatas - Preencha valores ausentes com a mediana - Padronize datas para YYYY-MM-DD - Remova outliers > 3 desvios padrao - Exporte para CSV limpo" IA: Gerando código de limpeza... import pandas as pd df = pd.read_excel('dados_brutos.xlsx') df = df.drop_duplicates() df['valor'] = df['valor'].fillna( df['valor'].median()) # ... [código completo gerado] df.to_csv('dados_limpos.csv')
Curso ABAR 2026 16 / 28
Limpeza consome 60-80% do tempo em data science. Com IA, voce descreve o que quer e ela gera o código. Mostrar um exemplo real de dados sujos e como a IA limpa automaticamente.
